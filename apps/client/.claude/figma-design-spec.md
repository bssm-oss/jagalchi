## 🎨 전체 레이아웃 구조

```
┌─────────────────────────────────────────────────────────────┐
│ Header (고정, 최상단)                                          │
│ "< Jagalchi Roadmap"                                         │
└─────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────┬──────────────────────┐
│                                      │                      │
│                                      │   Sidebar (고정)      │
│         Canvas (React Flow)          │   - 항상 열림         │
│                                      │   - 우측 고정         │
│                                      │   - 닫기 버튼 없음    │
│                                      │                      │
│                                      │                      │
└──────────────────────────────────────┴──────────────────────┘
               ┌──────────────────┐
               │   Toolbar (중앙)  │
               │   아이콘 4개 + ⚙️  │
               └──────────────────┘
```

### 레이아웃 특징

- **Header**: 왼쪽 상단 고정, "< Jagalchi Roadmap" 텍스트
- **Canvas**: 전체 화면, React Flow 캔버스
- **Sidebar**: **항상 열린 고정 패널** (우측), Sheet가 아님
- **Toolbar**: 하단 중앙, 떠있는 UI (floating), 4개 아이콘 + 톱니바퀴

---

## 🖼️ Header 디자인

```
< Jagalchi Roadmap
```

- 왼쪽 상단 고정
- "< " + "Jagalchi Roadmap" (뒤로가기 + 제목)
- 배경: 투명 또는 반투명 흰색
- AI 버튼 **없음** (AI는 툴바에서 진입)

## 🛠️ Toolbar 디자인

### 기본 구조

```
┌────────────────────────────────────┐
│  [노드] [선] [섹션] [텍스트] [⚙️]   │
└────────────────────────────────────┘
```

### 아이콘 목록

1. **노드 추가** (사각형 아이콘)
2. **선 추가** (화살표 아이콘)
3. **섹션 추가** (큰 사각형 아이콘)
4. **텍스트 추가** (T 아이콘)
5. **톱니바퀴 (⚙️)** - AI 기능 진입점

### 톱니바퀴 Dropdown

- **로드맵 생성**: AI Dialog 열림 (생성 탭)
- **로드맵 수정**: AI Dialog 열림 (수정 탭)

**중요**: AI 기능은 툴바 톱니바퀴에서만 진입 가능 (Header에 AI 버튼 없음)

## 📂 Sidebar 디자인

### 기본 특징

- **항상 열림** (Sheet가 아님)
- **우측 고정 패널**
- **닫기 버튼 없음**
- 선택에 따라 내용만 바뀜

### Sidebar 구조 (Node 선택 시 예시)

```
┌──────────────────────────────────┐
│ >> Node_2-2-2               [🔒] │  ← 제목 + 잠금 아이콘
├──────────────────────────────────┤
│ 노드 이름                         │  ← 섹션 제목
│ [Node_2-2-2           ]          │  ← Input
├──────────────────────────────────┤
│ 노드 설명                         │
│ [Desription of Node_2-2-2 ]      │  ← Textarea
├──────────────────────────────────┤
│ AI 생성                          │  ← 섹션 제목
│                                  │
├──────────────────────────────────┤
│ 기본 컬러                         │
│ [●][●][●][●][●]                  │  ← 5색 프리셋
├──────────────────────────────────┤
│ 커스텀                           │
│ [😀] [████████████]              │  ← 2D 그라디언트 피커
│ [━━━━━━━━━━━━━━━━]               │  ← HEX 슬라이더
├──────────────────────────────────┤
│ 형부 자료                         │
│ [자료 링크 입력        ]          │  ← Input
│ [자료 링크 입력        ]          │
│ [자료 링크 입력        ]          │
│ [+]                              │  ← AI 추천 버튼
│                                  │
│ AI 추천                          │  ← 버튼
└──────────────────────────────────┘
```

### Sidebar 타입별 내용

#### 1. Node Sidebar

- 노드 이름 (Input)
- 노드 설명 (Textarea)
- AI 생성 (버튼)
- 기본 컬러 (5색 프리셋)
- 커스텀 (2D 그라디언트 ColorPicker)
- 형부 자료 (Input 3개 + AI 추천 버튼)

#### 2. Line (Edge) Sidebar

- 선 (제목)
- 연결선 (라벨)
- 라벨 (Input, 예: "Worse")
- 스타일 (Dropdown: 실선/점선/파선)
- 꺽임선 (Checkbox)
- 화살표 (Slider, 0-1-2)
- 두께 (Input, px)
- 기본 컬러 (5색)
- 커스텀 (2D 그라디언트)

#### 3. Section Sidebar

- 섹션 (제목)
- 섹션 이름 (Input)
- 크기 (W, H Input, px)
- 기본 컬러 (5색)
- 커스텀 (2D 그라디언트)

#### 4. Text Sidebar

- 텍스트 (제목)
- 텍스트 크기 (Input, px)
- 기본 컬러 (5색)
- 커스텀 (2D 그라디언트)

#### 5. Multi-selection Sidebar (다중 선택)

**노드 다중 선택:**

```
┌──────────────────────────────────┐
│ >> 노드 다중 선택                  │
├──────────────────────────────────┤
│ 정렬                              │
│ [왼쪽정렬][중앙][오른쪽][상단][중간][하단] │  ← 6개 정렬 아이콘
├──────────────────────────────────┤
│ 간격                              │
│ [미이][미][Mixed        ]         │  ← Mixed 상태
├──────────────────────────────────┤
│ 노드 이름                         │
│ [Mixed                ]           │  ← 여러 값 혼합
├──────────────────────────────────┤
│ 노드 설명                         │
│ [Mixed                ]           │
├──────────────────────────────────┤
│ 기본 컬러                         │
│ [●][●][●][●][●]                  │
├──────────────────────────────────┤
│ 커스텀                           │
│ [😀] [████████████]              │
│ [━━━━━━━━━━━━━━━━]               │
└──────────────────────────────────┘
```

**정렬 아이콘 6개:**

1. 왼쪽 정렬
2. 수평 중앙 정렬
3. 오른쪽 정렬
4. 상단 정렬
5. 수직 중앙 정렬
6. 하단 정렬

**Mixed 상태:** 여러 요소의 값이 다를 때 "Mixed" 표시

## 🎨 ColorPicker 디자인

### 2D 그라디언트 ColorPicker (중요!)

```
┌────────────────────────────────┐
│ 기본 컬러                       │
│ [●][●][●][●][●]                │  ← 5색 프리셋 (원형 버튼)
├────────────────────────────────┤
│ 커스텀                         │
│ [😀] ┌──────────────────┐      │
│      │  ████████████████ │      │  ← 2D 그라디언트 영역
│      │  ████████████████ │      │     (빨강→흰색 그라디언트 예시)
│      │  ████████ ◯ █████ │      │     원형 선택자(◯)로 색상 선택
│      │  ████████████████ │      │
│      └──────────────────┘      │
│ [━━━━━━●━━━━━━━━━━━]          │  ← HEX 슬라이더 (색상 스펙트럼)
└────────────────────────────────┘
```

### 구성 요소

1. **기본 컬러 (5색 프리셋)**
   - Black (#000000)
   - Blue (#3B82F6)
   - Purple (#8B5CF6)
   - Red (#EF4444)
   - Orange (#F97316)

2. **2D 그라디언트 피커**
   - 가로: 색상 순도 (선택된 색 → 흰색)
   - 세로: 명도 (밝음 → 어두움)
   - 원형 선택자(◯)로 드래그하여 선택

3. **HEX 슬라이더**
   - 하단: 무지개 색상 스펙트럼 (빨강→주황→노랑→초록→파랑→보라)
   - 슬라이더 핸들(●)로 기본 색상 선택
   - 선택 후 2D 그라디언트에서 세부 조정

**기존 구현과의 차이:**

- ❌ HEX Input 텍스트 필드 없음
- ✅ 2D 그라디언트 영역으로 직관적 선택
- ✅ 하단 슬라이더로 색상 스펙트럼 선택

## 🤖 AI 기능

### 진입점

- **툴바 톱니바퀴(⚙️)** 클릭 → Dropdown 열림
  - "로드맵 생성" → AI Dialog (생성 탭)
  - "로드맵 수정" → AI Dialog (수정 탭)

### AI Dialog 구조

```
┌───────────────────────────────────────────────┐
│ ✨ 로드맵 생성                          [X]   │  ← 제목 + 닫기
├───────────────────────────────────────────────┤
│ [로드맵 생성] [로드맵 수정]                    │  ← 2개 탭
├───────────────────────────────────────────────┤
│ 로드맵 정보                                    │
│ 말이시                                         │  ← 작은 설명 텍스트
│                                               │
│ ┌───────────────────────────────────────────┐ │
│ │                                           │ │
│ │  (프롬프트 입력 영역)                      │ │  ← Textarea (큼)
│ │                                           │ │
│ │                                           │ │
│ └───────────────────────────────────────────┘ │
│                                               │
│                         [✨ 생성하기]          │  ← 버튼 (우측 하단)
└───────────────────────────────────────────────┘
```

### 로드맵 생성 중 (로딩 상태)

```
┌───────────────────────────────────────────────┐
│ ✨ 로드맵 생성                          [X]   │
├───────────────────────────────────────────────┤
│ [로드맵 생성] [로드맵 수정]                    │
├───────────────────────────────────────────────┤
│ 수정사항 정보                                  │
│ 이번 부분을 이렇게 수정할거 생각하고싶습니다.    │
│                                               │
│ ┌───────────────────────────────────────────┐ │
│ │                                           │ │
│ │  (생성 중 표시)                            │ │
│ │                                           │ │
│ │                                           │ │
│ └───────────────────────────────────────────┘ │
│                                               │
│                         [수정]                │  ← 버튼 (비활성화)
└───────────────────────────────────────────────┘
```

### AI 원본 자료 추천 (Popover)

노드 Sidebar의 "AI 추천" 버튼 클릭 시:

```
┌────────────────────────────────────────┐
│ ✨ 원본 자료 추천                   [X] │  ← 제목
├────────────────────────────────────────┤
│ AI 추천 자료                            │
│                                        │
│ [ResourceSite.com/Resource1        ][↗][+] │
│ [ResourceSite.com/Resource2        ][↗][+] │
│ [ResourceSite.com/Resource3        ][↗][+] │
│ [ResourceSite.com/Resource4/long...][↗][+] │
│                                        │
│                            [전료]      │  ← 버튼
└────────────────────────────────────────┘
```

- 각 자료 링크마다:
  - [↗] 외부 링크 열기 버튼
  - [+] 추가 버튼
- 하단: "전료" (닫기) 버튼
