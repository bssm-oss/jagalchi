# Do Not Touch

## Files Claude Should NOT Modify

- `pnpm-lock.yaml` - Let pnpm handle this
- `.env.local` - Contains secrets
- `next-env.d.ts` - Auto-generated by Next.js
- `.next/` - Build output

## Patterns to Avoid

- Do NOT delete existing tests without explicit permission
- Do NOT change ESLint/Prettier config without discussion
- Do NOT modify `components.json` (shadcn config)

## When Uncertain

Ask before:

- Changing project structure
- Adding new dependencies
- Modifying CI/CD workflows
